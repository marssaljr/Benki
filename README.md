## Installation
```sh
$ sudo apt install python3-pip
$ sudo pip3 install -r requirements.txt
```
## Usage
```sh
$ chmod + x benki.py
$ ./benki.py
```
Move mouse for position and use the keys 'Tab' and 'Space' to make a click
